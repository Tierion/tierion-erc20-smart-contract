### Setup

You'll need the [yarn tool installed](https://yarnpkg.com/lang/en/docs/install/).

Install dev dependencies:

```
cd tierion-erc20-smart-contract
yarn
```

### Test

Running `yarn test` will also take care of starting/stopping `testrpc`.

```
yarn test
```
